方法篇

第一节：“滚动学习”，走出传统学习思维误区

走出误区

  • 语言学习是靠天赋和智力决定的
    英语学习和智力一点关系没有！只要你重复学习，一个词练上千万遍，没有学不会的。
  • 初学英语先从语法开始
    你看小孩开始学说话的时候，有先从语法开始教授吗？都是先呀呀学语

三个基本态度

  • 善用零星时间
  • 少就是多，慢就是快
  • 持之以恒，永不放弃

基本学习过程

  • 采用滚动式学习法并少量内容学习
  • 强烈建议大家去自创一个学英语的环境

第二节：颠覆认知:回归正确英语学习顺序

学生学习英语过程的误区：

死记单词->死背课文->死记语法->应试考试->学单词题、阅读测试、完形填空、作文及翻译

今日不要脸，他日更有脸

“若想学好英语，首先就是不要脸！”
见到外国人就大胆说出脑子里想出来的字词，尽管词不达意，语法错误都没关系。能够从自己嘴里迸出英语出来就已令他们敬佩。

自学英语的四个过程

  • 文字
  • 语法
  • 规范
  • 听

学习英语的态度：

“过度学习”（overlearning）— 也就是重复练习 可提升我们的持久学习记忆能力，我们越熟悉某个技巧，就越能融会贯通其它的技能并臻于至善。

秘诀：

我们学习语言的秘诀就是：善用零星时间。
不论在学听、说、读、写，都要一遍又一遍地学习，时间久了就能熟能生巧。
所以，学习英语的四个过程是：说 → 读 → 写 → 听

第三节：巧用工具，互联网学习的6个方法

如何利用互联网学习英语听说读写的能力?

发音

  • rachelsenglish

推荐原因:

  • 1.美籍老师，地道美式发音及夸张的嘴型，让学者很容易跟着学习。
  • 2.节目附有影像，采用 the audio-visual approach（影像同步教学法），方便学习者模仿学习

学习方法小贴士：

  • 1.每天只学两个元音，学完后再学辅音。
  • 2.要求自己每天坚持模仿这些元音几千遍以上，使发音的嘴型与舌头的位置能学好“一次到位”。

基础会话

  • Basic English Conversation
  • Daily English Conversation

选择小技巧：

  • 1.选择有英语字幕的节目。附带有情境录像的教学，反而令学习者分心。
  • 2.选择会话教材学习时，尽量挑有真人发音、语调自然且发音极准的教材，这样适合读者轻声跟读模仿。

模仿的技巧（shadowing）的好处：

  • 1.不知不觉中提升英语对话的理解能力；
  • 2.不知不觉中提升自己说英语的速度；
  • 3.不知不觉中成功模仿该国人自然说话的语调；
  • 4.不知不觉中大幅改善自己的单词短语发音及连音。

学习方法小贴士：
每则对话的内容并跟读几百次后，采用“角色扮演”，把会话的内容一人饰两角表演出来，不要死记内容。
每天做到两三百遍以上，其学习效果惊人！

学听慢速有声的美语读本

  • 国外公司精良制作的有声美语读本文字及音文件

    学习方法小贴士：
    一次购买一本，完全弄懂全文意思及语法盲点后，用shadowing 的方式跟读，最后用自己的英语诠释全书内容，一遍一遍地诠释，直至诠释得流畅无阻为止。

  • 电台节目《美国之音》互联网电台专为全球各国想学好美语听力的读者所设的特别节目：of America Special English
    选择原因：
    完全免费且内容每天都会更新。
    每日报导国际间发生的重大新闻，由专业的男女播音员播报，口齿咬字清晰，且语调上抑扬顿挫，最适合初学听力的人模仿跟读。

学习方法小贴士：

  1. 一天只听一则新闻，再看文字以及查字典，轻声跟读（shadowing）两三百遍，再用诠释（paraphrasing）的方式重新诠释内容
  2. 每晚在睡前用二十分钟通过手机或计算机 playback（回 放）的方式逐字听写。

选择免费的Cambridge Dictionary

辞典中所有的单词都是英英和英汉双解，并附实用例句及相关语法。对英语初学者而言，Cambridge Dictionary 所有单词都附有英国人或美国人的真人发音音频，较为实用。

可上网选择看： USA TODAY 《今日美国》

选择原因：

  • 1.日报的文章用词简练，不卖弄文字。
  • 2.报纸同时也播报视频新闻，除了学习英语阅读能力外，也提供了正常语速学习英语听力的渠道，提升高级听力。

学习方法小贴士：

  1. 多听谈话性的节目，可以熟悉不同的英语口音腔调。
  2. 国内 CCTV 中央电视台英语频道节目，受邀采访的对象来自全球各地各行，说英语的腔调各有不同，对提高听力及口译能力有很大的帮助！

口语篇

第四节 “过度学习法” 说出地道的美式口语

“要说一口漂亮的英语得先从正音开始，也就是一定要学会地道美式K.K.音标的发音”

我只利用三个月的时间采用 overlearning（过度学习）的方式，凭着听一张 K.K.音标教学唱片学会了音标。也学会了即兴自由开口说英语。

我是如何学习“说”英语的？

1.要从口语学起，要活记单词，而不是死记单词。
2.学习口语的首要条件是寻找正确的声音源

三个基本态度

  • 1.善用零星时间
  • 3.持之以恒，永不放弃
  • 2.少就是多，慢就是快

学习方法小贴士：

  • 1.每天只学两个元音，学完后再学辅音。
  • 2.要求自己每天坚持模仿这些元音几千遍以上，使发音的嘴型与舌头的位置能学好“一次到位”。

把自己想成是新生儿，每天只学两个音标

过度学习的功效Overlearning（过度学习） 让我注意到重复模仿的重要性。
学了三个月的音标后，我看到字典中的音标，就能立刻发出和美 国人一模一样的单词读音了。

学元音需要注意的细节

a

  • [ i ] 念法：应念成类似“益”或“易”的音，属长元音。
  • [ I ] 念法：应念成军人行进时打数“一、二、三、四”的“一”声。

b

  • [ E ] 念法：类似罗马拼音的 e 音，就像羊“咩咩” 叫的元音。
  • [ A ] 念法：发 [ A ] 音时要比 [ E ] 夸张，即嘴巴要尽量张得宽。

c

  • [ u ] 长元音。念法：念类似普通话“雾”或“物”的发音，但不尽相同，念 [ u ] 时，舌根应较紧，嘴唇要更翘一些。
  • [ U ] 短元音。念法：发“ʊ”的音时，先将嘴念成短促的“雾”或“物”声，再念成“耳”声，听起来像 “雾耳”或“物耳”的音。

举例：boo k[bʊk] good[gʊd]
这两个单词如果念成“布可 ”、“固的”，是错的。正确念法：“布尔可 ”（“ 布”不发声）或“固耳的”（“的”要发声，但几乎听不 见）。

d

  • [ ZU ]或 [ Zʊ ] 双元音。念“傲”的音，其实是错误的。正确念法：先将嘴全张，先念成类似“阿”的音，再念类似“傲”的音，最后才念类似短“雾”的音，一气呵成，即“阿—傲—雾”。

举例：Mountain [‘mZʊntN ] downtown [dZʊn’tZʊn]这两个单词如果 念成“ 忙藤 ”或“ 当烫 ”就 错了， 因为你念“ 忙”或“ 当烫 ” 时，嘴巴并没完全张开。正确念法：念成 类似 “妈阿嗯藤”或“大昂踏昂 ”的音。

e

  • [ V ]与[ J ] 念法：发音相同，念成类似普通话的“饿”发音。只不过 [ V ] 是强音，[ J ] 是弱音。
    举例：

  • 1.[V]出现在 单音 节或重音 节中， 如：cut [kVt] mu g [mVg]

  • 2.[V]也出现在 重音 节中， 如：above [J’bVv]moth er [‘mVTK ]

  • 3.[J]则出现在 有两个音节以上的单词中的弱音节中，如：above [J’bVv] a bout [J’bZʊt]

f

[ R ] 与[ K ] 念法：发音相同，均念成类似普通话“饿尔”的发音。不过念“尔”时要卷舌。[ R ] 出现在单音节或重音节中，[ K ] 则出现在弱音节中。

举例：girl sist er matt er mu rder[gRl] [‘sIstK][‘mAtK][‘mRdK]

g

  • radio 瑞丢（X ）video 威 丢（X）ruì diūwēi diūRadio [ ‘redI“o ] （类似“瑞 底 欧”）下列两个单词千万不要念成两个音节：wēi dǐ ōu ruì dǐ ōuVideo [ ‘vIdI”o ] （类似“威 底 欧”）

学辅音需要注意的细节

a

辅音s起首形成的字之后有字母 p、t、k 时，一定要将 p、t、k 念成 [ b ]、[ d ]、[ g ]。

如下：

字典列出的音标 实际要念成
speak [ spik ] [ sbik ]
stop [ stap ] [ sdap ]
skirt [ skRt ] [ sgRt ]

注意:
如果是两音节的单词，第二音节以 s 起首时,
之后的 p、t、k 一样念成 [ b ]、[ d ]、[ g ]

如下：

字典列出的音标 实际要念成
sister [ ‘sIstK ] [ ‘sIsdK ]
whisper [ ‘wIspK ] [ ‘wIsbK ]
whisky [ ‘wIskI ] [ ‘wIsgI ]

b

字典列出两个音节以上的音标，尾音有 [ tI ]、[ dI ]、[ sI ]、[ kI ]、[ mI ]、[ nI ]、[ gI]、[ djI ]、[ bI ] 等时，一律将 [ I ] 念成长元音 [ i ]。

如下：

字典列出的音标 实际要念成
Bobby [ ‘babI ] [ ‘babi ]
tidy [ ‘tZIdI ] [ ‘tZIdi ]
Tammy [ ‘tAmI ] [ ‘tAmi ]
sissy [ ‘sIsI ] [ ‘sIsi ]
whisky [ ‘wIskI ] [ ‘wIsgi ]

注意：如今许多英英字典都将上列尾音的[ I ]改为[ i ]，可参考网络上“牛津”、“剑桥”这两本权威字典的真人美式发音。

c

在 K.K. 音标中，[ y ] 及 [ L ] 均属辅音，故这两个音标在一起时，不可连读。
一律将 [ I ] 念成长元音 [ i ]。

如下：

million [ ‘mIlyJn ] 应念成类似普通话的“没有眼”（méi yǒu yǎn），而非“咪淋”（mī lí）。nbillion [ ‘bIlyJn ] 应念成类似普通话的“比有眼”（bǐ yǒu yǎn ），而非“比淋”（bǐ lí ）。línvalue [ ‘vAlyu ] 应念成类似普通话的“维埃欧有”（wéi āi ōu yǎn），而非“维埃欧溜”（wéi āi ōu liū）。

注意:
念“维”时，上排牙齿要咬住下唇，振动出声并发出爆破音。

d

凡是单词以辅音 [ t ]、[ d ]、[ b ]、[ p ]、[ k ]、[ g ]结尾时，美国人以正常速度念这些单词时，这些辅音通常在作势要念出来时立即切除，改以短暂停顿代替。

如下：

字典列出的音标 实际要念成
want [ want ] [ wan* ]
wanted [ ‘wantId ] [ ‘wantI* ]
sat [ sAt ] [ sA* ]
sad [ sAd ] [ sA* ]
sag [ sAg ] [ sA* ]
bib [ bIb ] [ bI* ]
bit [ bIt ] [ bI* ]
bite [ bZIt ] [ bZI* ]
keep [ kip ] [ ki* ]
kick [ kIk ] [ kI* ]

e

很多有 [ yu ] 的音标，在英式英语会照念，但在美语则省略 [ y ]。
如下：

英式英语 美语
student [ ‘styudJnt ] [ ‘studJnt ]
news [ nyuz ] [ nuz ]
due [ dyu ] [ du ]

f

美语中，两音节的单词结尾音标 [ -dJnt ]、[ -dJn ]、[ -tJn ] 改成 [ -dNt ]、[ -dN ]、[ -tN ]，表示这些音标组合都发鼻音，类似张嘴念“嗯”的音（不可闭嘴）

如下:

mountain [ ‘mZUntN ] 实际念成“妈昂嗯”(mā áng èn cotton)。
cotton[ ‘katN ] 实际念成“卡嗯”(kǎ èn)
student [ studN* ] 实际念成“死都嗯”（“死”无声音）(sǐ dōu èn)

g

有些英英字典出现斜体的 [ J ]，表示在英语中，这个元音不需念出来。

如下：

字典中列出的音标 实际念成
comfortable [ ‘kVmfJtJbL ] [ ‘kVmftJbL]
sophomore [ ‘safJ”mOr ] [ ‘saf”mOr ]
（注：* [ bL ] 是 [ bJl ] 的简写）

h

以 -dly或 -tly结尾的单词，d 或 t 应作势要发成 [ d ] 或 [ t ] 的辅音，但实际立即停顿，不发出音来。

如下：

字典中列出的音标 实际念成
immediately [ I’midIJtlI ] [ I’midIJ*li ]
friendly [ frEndlI ] [ frEn*li ]

第五节 3个月英语对话无压力 “自我对话法”

三个月学完音标后，我进入到我学习的第二个过程：学习会话。从不会开口到可以用简单英语与外国人沟通，也是在短短三个月完成的。

去英语培训班学习可靠吗？

不建议培训班的原因：

  • 1.补习费价格不菲，学习英语的成本高。
  • 2.你不知道外籍老师到底来自哪个美国区域。美式英语因地区的不同，口音也不一样。
  • 3.一个班至少十人以上，真正轮到你讲英语，或跟老师互动的时间绝对是少之又少。

建议自我学习：
自我学习英语会话可在任何无人的场所进行，且不花一文钱，一举数得，何乐而不为？希望大家记住：重复，重复，再重复。学英语就是一个重复的过程，也就是overlearning（过度学习）态度！

自学英语会话的两个方式

  1. learned to talk to myself （我自己跟自己讲英语）
  2. did role - playing games（我做角色扮演）

学习方法：

  • 1.一天学一篇会话，滚动式学习，一天说上千遍上万遍。
  • 2.把一篇会话看懂听懂后，一直不断慢速地去念。
  • 3.完全掌握会话内容后，开始一遍一遍地凭印象把会话内容以一人分饰两角的方式表演出来。

研读会话步骤

  • 1.先查字典，确定每一个字发音都要标准；
  • 2.慢速念会话里的英文句子，边念边想着中文意思；
  • 3.念了几百遍后，把文章丢掉继续念会话，你会忘记部分原文或者做些修改，再结合之前练习过的会话，变成以下一则你自创的会话
  • 4.你延伸了一些内容，之后再延伸一些内容。就这样慢慢讲并不断延伸会话，变成你的东西。过去在各则会话中学到的知识，未来会慢慢地拓展，变成了更较长的对话。
  • 5.这就变成你可以自言自语讲好多会话，渐渐地这些英语单词、短语、词组，甚至句子就变成你的东西，每一次你讲英语时自然就会运用出来，在不知不觉的状态中运用自如。

我如何练习每天，自学的这些英语

  • 1.发现我说英语越来越顺畅后，特意去找有人可以跟我讲英语的地方，找到机会练习我每天自学的这些英语。
  • 2.找一个有很多讲英语高手的场合加入，可以听到他们是怎么讲英语的，也可以有机会定期与其他人用英语交谈。

学习英语的三个基本态度：

  • 1.善用零星时间
  • 2.少就是多，慢就是快
  • 3.持之以恒，永不放弃

阅读篇

第六节 “阅+读”英语能力进阶提升

我如何学习“读”英文？

  • 1.大量阅读文章，扩展词汇量
    想从简易的会话书中增加词汇量，使英语提升到中文水平的地步，这是不可能的事。
    增加词汇量方法：
    长期停留在学会话的阶段，所学的都是一些简短会话里的生活用语，词汇量的增加必然会受到限制。
    因此我决定要大量阅读英语文章，以扩展我的英语谈话内容，更要提升我对谈话主题的深度
    读什么英语文章好？

    • 《 Reader’s Digest 》

      中文版《读者文摘 》—内容是由长篇文章浓缩的（condensed [ kJn’dEnst ]）a condensed article 就是“浓缩的文章”。
      选择原因：

        1. 一篇优秀浓缩的文章言简意赅，有很高的文字素养。
        1. Reader’s Digest 里面文章主题丰富多样，内容涵盖现代社会一切生活英语方面的知识。
    • 《 China Post 》

      《英文中国邮报 》—journalistic English（新闻英语）是给大众看的，相当于是英美国人的初中学习程度。
      选择原因：

        1. 用词简洁明了，写作风格严谨，句子合乎语法。
        1. 新闻报 导的内容 包罗万象，比Reader ‘s Digest 更广，从英文报纸可以学到有关政 治、 经济、体育 、娱乐 、文学、艺术 、科技等关键词汇。

    英语文章学习方法：

    1. 看到英语文章就要开口念出声音来，而不是只默读， 也不是马上查字典。
    2. 一小段一小段地攻克文章而不是一次一整篇念完 ，“少就是多”原则。

    大量阅读的方法习惯是：
    大量地“阅”与“读”—这里 讲的阅读是“阅”与“读”。也就是 说，当 你的眼睛“看”到英语文章时，你的嘴巴 就必 须要同时把 英文 句子“念”出来 。

第七节 “8步阅读法”阅读词汇同步增长

第二步

通体浏，知道大概意思。

第二步

第一天看懂第一段,第二天看懂第二段少就是多，一天看一段

第三步

第一遍，慢速念，避免pause（停顿）。
记住第一遍：整体通读，不要停，利用上下文，整体理解。

第四步

第二遍：逐句查，音标、字义、字词的用法及例句都要查，抄笔记。

第五步

将第一段新闻译成中文。

第六步

利用零星时间，慢速念英文，同步翻译成中文，反复训练，久而久之，理解力变强，速度变快。

第七步

睡觉前，我再翻开笔记本将单词及用法从头到尾念一遍，念完后就用这些单词及用法自己造句，尽量多说。

第八步

第二天，同样的方法看第二个段落；第三天，同样的方法看第三个段落；………以此类推 ，直到读完整篇报道 。久而久之，少就变多了，养成 monologue（独白）的习惯。第一年，看一份报纸 ，但少就是多，学得很扎实；第二年，速度加快 ，一年用这样的方式可以看完好几份报纸了。

第八节 “阅读实操课” 掌握阅读技巧

第一步

通体读完

第二步

第一天学习第一段，第二天学习第二段……依此类推

第三步

慢速读完，整体理解这一篇的大意。
切记：
慢慢念 ， 且绝不马上查不熟的单词 ，先抓住这段的中文大意

第四步

逐句查，音标、字义、字词的用法及例句都要查，抄笔记。

第五步

进行翻译

第六步

简单的英语诠释。
不断重新诠释 ，直到可以灵活发挥

第七步

睡觉前, 翻开笔记本，将单词和用法从头到尾念一遍，再自己造句。如使用单词 attack 及 make an attack on 造句。

第八步

这就是我养成大量地“阅”和“读”的方法，让学习变得事半功倍：

  • (1 )单词量增加 。
  • (2 )因为要勤查字典，所以学到了字词的collocation（字词搭配 ） ，而这种字词搭配 ，其实这就是 我之后要谈到的语法 。
  • (3 )能看懂英英字典，到二年级下学期时我就改成使用英英字典了。
  • (4 ) 奠定我的独白能力 。

能够获致这样的成果，是因为中间历经了前面说的两个阶段：
第 一 ：学发音学会话。
第 二 ：大量地“阅”与“读”。

经历的这两个阶段，为第三阶段“写作”奠定了坚实的基础。从阅读报纸中得到许多好处。

第九节 “5大类动词”全面覆盖英文读写

我如何学习“读”英文？

  • 1.可做主语的词类、短语、词组或从句
    • （1）名词
      That young boy is my brother.（那个年轻的小伙子是我弟弟。）
    • （2）代词
      He enjoys singing.（他喜欢唱歌。）
    • （3）动名词
      即动词变成的名词（动词原形 + ing），表示自己曾经做过或已知的经验。中文可用动词做主语。如下
    • （4）不定式短语
      即“to + 动词原形”（如 to sing、to leave、to buy something），用以表示自己尚未做过或不一定在何时才会做的一件事。中文可用动词做主语
    • （5）名词从句
      即句子变成的名词。中文可用句子做主语，英语则不行
      名词从句一共有三种：
      • （a）由陈述句变化而成：
        陈述句：John is honest.（约翰很诚实。）
        名词从句：that John is honest
      • （b）由一般问句（即可用 Yes / No 回答的问句）
        一般问句：Can Mary do it?（玛丽能做这件事吗？）
        名词从句：whether Mary can do it
        一般问句：Is the answer correct?（那个答案是正确的吗？）
        名词从句：whether the answer is correct注意.
      • （c）由特殊疑问句（即以疑问词如 what、which、where、how、who、when、why 等起首的问句）变化而成：
        特殊疑问句：Where does that girl live?（那个女孩子住在哪儿？）
        名词从句： where that girl lives
        特殊疑问句：Why is she crying?（她为什么在哭？）
        名词从句： why she is crying
    • （6）名词词组
      这种名词词组都是由疑问词（如 what、which、where、how、who、when 等疑问词）接不定式短语形成
  • 2.动词的种类及功能

    动词是句子的心脏，如果句子中没有动词，就无法形成一个句子。只要有动词，即使没有主语，也可形成祈使句

    例句：
    Come here! （到这儿来！）
    Be quiet! （安静！）
    Go to bed now. （现在就去睡觉。）
    动词有五大类：
    ● 完全及物动词
    ● 完全不及物动词
    ● 不完全及物动词
    ● 不完全不及物动词
    ● 授予动词

判断五大动词的方法：

  • 完全及物动词

    1. 我先把一个动词如 punish（处罚）放入两个中文句子的空格中：
      我　　 punish　　 他。
      他被我 punish　　。
    2. 再将这两句翻译成中文如下
      我  处罚  他。
      他被我  处罚   了。
    3. 如此一来，两句中文意思都很通顺，而且意思很完整，则 punish 是意思很完整的及物动词，它就是完全及物动词。

    完全及物动词在句中出现的基本形态有下列几种：
    (a) 主语 + 完全及物动词 + 宾语（主动语态）
    (b) 主语 + be 动词（is、was、were、has been、have been）+ 完全及物动词的过去分词形（被动语态）
    (c) 主语 + be 动词（is、was、were、has been、have been）+ 完全及物动词的现在分词形 + 宾语（进行时）

    完全及物动词也有进行时的被动语态，惟其使用频率较无上述三种用法常见，即为主语 + be 动词 + being + 完全及物动词的过去分词形。

    完全及物动词也有进行时的被动语态，惟其使用频率较无上述三种用法常见，即为主语 + be 动词 + being + 完全及物动词的过去分词形，例：

  • 完全不及物动词
    完全不及物动词，这种动词在句中出现的基本形态只有两种：
    (a) 主语 + 完全不及物动词（主动语态）
    (b) 主语 + be 动词（is、was、were、has been、have been）+ 完全不及物动词的现在分词形（主动语态进行时）

  • 不完全及物动词
    动词 go、finish、sell out 就可采过去分词形置 be 动词后做形容词
    从上面的第一个英语句中，我们得知 named（将……取名为）是及物动词，之后接 the little boy 做宾语，再接名词 John 补充句意的不足。此时我们就明白 named 是意思不完整的及物动词，即不完全及物动词，John 则称为补语，修饰宾语 the little boy，故亦称为宾语补语。

主语的词类、词组或从句，以及动词的五大类：

  • 完全及物动词 （complete transitive verb，简称 vt.）
  • 完全不及物动词（incomplete transitive verb，简称 vt.）
  • 不完全及物动词（complete intransitive verb，简称 vi.）
  • 不完全不及物动词（incomplete intransitive verb，简称vi.）
  • 授予动词（dative verb，属“及物动词”，简称vt.）

英语的五大基本句型

  1. 主语 + 完全及物动词 + 宾语
    I    ‘made’（完全及物动词）  ‘a chair ‘（宾语） yesterday.
    （我昨天制作了一张椅子。）
  2. 主语 + 不完全及物动词 + 宾语 + 补语
    We all    call（不完全及物动词）     him（宾语）   a liar（补语） .
    （我们都称他为骗子。）
  3. 主语 + 不完全及物动词
    John    works（完全不及物动词）      very hard （副词（修饰works）.
    （约翰工作很努力。）
  4. 主语 + 不完全及物动词 + 补语
    Mr. Li    is（完全不及物动词）      an English teacher （补语）.
    （李先生是英语老师。 ）
  5. 主语 + 授予动词 + 间接宾语 + 直接宾语
    I bought（vt.） （my little son ）（间接宾语） （a toy）（直接宾语）  （as his birthday present ）（介词 as 引导的词组（做形容词用，修饰 a toy））.
    = I bought（vt.） a toy（做bought的宾语）  for my son（介绍for的宾语） as his birthday present.
