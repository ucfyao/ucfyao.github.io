linux和windows协同合作的samba服务器配置

下载 centos yum install samba yum install samba-client ubuntu sudo apt-get update sudo apt-get install -y samba samba-client 下面我们来共享群组可读写文件夹，假设你要共享的文件夹为： /home/ray/share 首先创建这个文件夹 sudo mkdir -p /home/ray/share sudo chmod 777 /home/ray/share 备份并编辑smb.conf允许网络用户访问 sudo cp /etc/samba/smb.conf /etc/samba/smb.conf_backup sudo vim /etc/samba/smb.conf 搜寻这一行文字 ; security = user 用下面这几行取代 security = user username map = /etc/samba/smbusers 将下列几行新增到文件的最后面，假设允许访问的用户为：ray。而文件夹的共享名为 Share //\[共享名\] \[Share\] //说明：comment是对该共享的描述，可以是任意字符串。 comment = Shared Folder with username and password //path用来指定共享目录的路径。可以用%u、%m这样的宏来代替路径里的unix用户和客户机的Netbios名，用宏表示主要用于\[homes\] 共享域。 path = /home/ray/share //说明：public用来指定该共享是否允许guest账户访问。 public = yes //说明：writable用来指定该共享路径是否可写。 writable = yes //说明：valid users用来指定允许访问该共享资源的用户。 valid users = ray create mask = 0755 directory mask = 0755 //说明：available用来指定该共享资源是否可用。 available = yes //说明：browseable用来指定该共享是否可以浏览。 browseable = yes 然后顺便把这里改一下，找到［global］把 workgroup = MSHOME 改成 workgroup = WORKGROUP display charset = UTF-8 unix charset = UTF-8 dos charset = cp936 后面的三行是为了防止出现中文目录乱码的情况。其中根据你的local，UTF-8 有可能需要改成 cp936。自己看着办吧。 现在要添加 ray 这个网络访问帐户。如果系统中当前没有这个帐户，那么 sudo useradd ray 要注意，上面只是增加了 ray 这个用户，却没有给用户赋予本机登录密码。所以这个用户将只能从远程访问，不能从本机登录。而且samba的登录密码可以和本机登录密码不一样。 现在要新增网络使用者的帐号： sudo smbpasswd -a ray sudo vim /etc/samba/smbusers 在新建立的文件内加入下面这一行并保存 ray = "network username" 如果将来需要更改 ray 的网络访问密码，也用这个命令更改 sudo smbpasswd -a ray 删除网络使用者的帐号的命令把上面的 -a 改成 -x //测试Samba的设置是否正确无误。 sudo testparm //重新启动 sudo /etc/init.d/smb restart ubuntu service smbd restart 最后退出重新登录或者重新启动一下机器。 cent OS7是通过systemctl 工具来启动了，方法如下： #systemctl enable smb.service #systemctl enable nmb.service #systemctl restart smb.service #systemctl restart nmb.service 可以查看安装包信息 rpm -qa | grep samba 假如直接linux直接连接linux的samba可参考man和–help smbclient -L //127.0.0.1 -U ray 这样可以列出你可以访问的目录及文件 smbclient //127.0.0.1/ray -U ray 这样了可以访问你的个人目录了 smbclient -L //127.0.0.1 -N 列出不须密码可以访问的目录及文件 smbclient //127.0.0.1/Document -N 匿名访问 Document目录 参考链接：http://www.cnblogs.com/xiaoluo501395377/archive/2013/04/06/3001863.html
